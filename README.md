# Verlet Integration - using Raylib

Currently has no optimization, will be optimized in future.

To compile and run the simulation, use in the project directory itself. No need to cd into `src`

Make sure Raylib is installed using a package manager or installed using `make install`

```shell
./build.sh
```

Note: `Raylib` is developed by raysan5. check [Raylib](https://github.com/raysan5/raylib) on github
